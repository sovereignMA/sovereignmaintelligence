Connect Google Ads

Pull campaign spend, impressions, clicks, and conversion data directly from Google Ads into this dashboard.

1Create a Google Ads API developer token in your MCC account → Tools → API Centre
2Create OAuth2 credentials for a service account with https://www.googleapis.com/auth/adwords scope
3Set env vars: GOOGLE_ADS_DEVELOPER_TOKEN, GOOGLE_ADS_CLIENT_ID, GOOGLE_ADS_CLIENT_SECRET, GOOGLE_ADS_REFRESH_TOKEN, GOOGLE_ADS_CUSTOMER_ID

Connect Google Analytics 4 Data API

GA4 tag G-9BQYFMVFTJ is already firing on all pages. Connect the GA4 Data API to pull sessions, events, and conversion funnels into this dashboard.

1Go to Google Cloud Console → enable Google Analytics Data API
2Create a service account → download JSON key → add the service account email as Viewer in GA4 Admin → Account Access
3Set env vars: GA4_PROPERTY_ID (numeric, e.g. 123456789), GA4_SERVICE_ACCOUNT_KEY (JSON string)

Connect TikTok Ads Manager

Pull TikTok campaign spend, video views, clicks, and conversions into this dashboard via the TikTok Marketing API.

1Go to TikTok for Business → Marketing API → apply for access and create an app
2Generate a long-lived access token with ad.read permissions
3Set env vars: TIKTOK_ACCESS_TOKEN, TIKTOK_ADVERTISER_ID

Connect Reddit Ads Manager

Pull Reddit campaign metrics — impressions, clicks, CPM, and conversion events — directly into this dashboard.

1Go to Reddit Ads Manager → create a developer application at reddit.com/prefs/apps
2Use client credentials OAuth2 flow (or user token with ads:read scope)
3Set env vars: REDDIT_ADS_CLIENT_ID, REDDIT_ADS_CLIENT_SECRET, REDDIT_ADS_ACCOUNT_ID
