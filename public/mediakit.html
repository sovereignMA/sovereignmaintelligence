About: Project Sovereign is an AI-native M&A command engine purpose-built to identify, approach, acquire, and exit UK SaaS businesses with zero personal capital and zero personal guarantees. Built by Howard Henry (solo founder, London) in a single weekend using Claude. The platform deploys 21 specialist AI agents across the full deal lifecycle — from sourcing to exit — backed by a 33-table PostgreSQL database, 8 Supabase Edge Functions, 8 automated cron jobs, and AES-256 encryption throughout.
